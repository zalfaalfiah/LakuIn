# Aplikasi LakuIn
## Aplikasi To-Do List untuk memenuhi Ujian Akhir Semester Mata Kuliah Pemrograman Mobile Multiplatform

Praktikum ini bertujuan untuk memahami dasar React Native, termasuk pembuatan komponen, penggunaan state dan props, layout dengan Flexbox, serta menampilkan data menggunakan FlatList.

## Fitur Aplikasi

- Add Task: menambahkan tugas baru dengan judul, tanggal, dan deskripsi
- Edit Task: mengubah data tugas yang sudah ada
- Delete Task: menghapus tugas dari daftar
- Date Picker: memilih tanggal tugas menggunakan kalender
- Validation: memastikan semua data terisi sebelum disimpan
- Navigation: berpindah antar halaman (Home, Add, Detail)
- State & Props: mengelola dan meneruskan data antar komponen
- React Hooks: menggunakan useState dan useEffect untuk logika aplikasi

## Teknologi
Aplikasi ini dibuat menggunakan:
- React Native
- Expo (versi 6.3.12)
- Node.js (versi 22.20.0)
- npm (versi 10.9.3)
- Supabase API

## Cara Menjalankan
1. Clone repo:
bash
git clone <https://github.com/zalfaalfiah/LakuIn.git

2. Masuk ke folder proyek:
bash
cd LakuIn

3. Jalankan proyek Expo:
bash
npx expo start

4. Pilih jalankan di emulator Android (Android Studio) atau scan QR lewat aplikasi Expo Go di HP.
